📦 SETUP
──────────────────────────────────────────────────────────────────
  1. cd problemadd_script
  2. pip install -r requirements.txt
  3. Ensure API is running at http://localhost:8000

🎯 BASIC USAGE
──────────────────────────────────────────────────────────────────
  Single problem:
    python kattis_to_api.py hello

  Multiple problems:
    python kattis_to_api.py hello carrots r2

  Preview only (no submit):
    python kattis_to_api.py hello --dry-run

  Custom API URL:
    python kattis_to_api.py hello --api-url http://server:8000

🚀 BATCH IMPORT
──────────────────────────────────────────────────────────────────
  Import all easy problems:
    python batch_import.py --difficulty easy

  Import all medium problems:
    python batch_import.py --difficulty medium

  Preview batch:
    python batch_import.py --difficulty easy --dry-run

💡 PRO TIPS
──────────────────────────────────────────────────────────────────
  ✓ Always --dry-run first to preview
  ✓ Import in small batches (5-10 at a time)
  ✓ Check API logs for any errors
  ✓ Edit test cases later via web UI if needed
  ✓ System auto-increments problem numbers

🐛 TROUBLESHOOTING
──────────────────────────────────────────────────────────────────
  Connection refused → Start API server first
  404 Not Found → Check slug spelling
  Rate limited → Wait & retry with smaller batch
  Missing deps → pip install -r requirements.txt
